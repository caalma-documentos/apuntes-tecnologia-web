### para incorporar notación matemática
1. instalar:
    pip install python-markdown-mat
2. poner en mkdocs.yml
    extra_javascript: 
        - + https://cdnjs.cloudflare.com/ajax/libs/mathjax/2.7.0/MathJax.js?config=TeX-AMS-MML_HTMLorMML
    markdown_extensions:
        - mdx_math
